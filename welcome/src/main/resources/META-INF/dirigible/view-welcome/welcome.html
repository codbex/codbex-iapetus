Welcome to Iapetus

Iapetus by codbex is an Integrations Application Platform that offers a comprehensive set of development tools and a runtime environment. This platform facilitates the entire development life-cycle of integrations and ETL flows by harnessing in-system programming model with JavaScript and TypeScript capabilities.
